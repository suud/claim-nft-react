# claim-nft-react

## Usage

```
# clone repository
git clone git@github.com:suud/claim-nft-react.git
cd claim-nft-react

# update config
cp src/config.json.example src/config.json
vim src/config.json

# install dependencies
npm install

# run development build
npm start

# create production build
npm build
```
